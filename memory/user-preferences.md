## Shopping & Online Orders

- **Verify all links before posting** — use WebFetch to confirm HTTP 200. Never post 404s.
- **AliExpress links expire** — re-verify on every check-in. If 404, find alternative with WebSearch.
- **Link format**: `✅ [Product Name - Price](url) - Verified YYYY-MM-DD`
- Prefer sellers with **4.5+ star ratings** and 100+ reviews
- Prefer **free shipping** when available (especially AliExpress Choice items)
- **Never post unverified links** — if you can't verify, say so and explain why

## Trello Updates (ALL sessions)

- **NEVER edit card descriptions** — they are user-maintained, read-only
- **ALL updates go in COMMENTS only**
- One comment per check-in; don't repeat the same information in consecutive comments
- When done: post a final summary comment with outcome

## Research

- Hebrew and English sources are both fine
- For Israeli services: check gov.il, official org sites first
- For products: include price in ILS (convert if needed, state conversion date)
- For medical/insurance: cite the official policy document or hotline number
- Preferred search: WebSearch first, then WebFetch to verify

## General

- Timezone: Israel Standard Time (IST / UTC+3)
- Currency display: ILS primary, other currency in parentheses
- Language for comments: match the card's language (Hebrew cards → Hebrew comments)
